== Description ==

Displays random verse from the holy quran.

== Installation ==

1. Upload the folder (quran) to the plugins directory (wp-content/plugins) or install it from your dashboard.
3. Activate the plugin from 'Plugins' menu in WordPress.
4. Go to Widgets page, and you'll see a new widget "Quran" add it to the region you prefer it to show up in and "Save Changes".
5. You can customize the look and feel using css/style.css inside the widget's folder.
